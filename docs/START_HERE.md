# Start Here

Welcome to GlyphStudio. This guide gets you productive in under five minutes.

## Quick Start Paths

### I want to draw a sprite
1. Launch GlyphStudio
2. ProjectHome &rarr; **Static Sprite** recipe
3. Draw with `B` (pencil), `G` (fill), `E` (eraser)
4. Export &rarr; PNG

### I want to animate
1. ProjectHome &rarr; **Animated Loop** recipe
2. Draw frame 1
3. `Ctrl+D` to duplicate, edit next pose
4. `Space` to play, `Left/Right` to step
5. Export &rarr; GIF

### I want to make variant sprites
1. ProjectHome &rarr; **Variant Family** recipe
2. Draw your base asset
3. Create palette sets in the palette workspace
4. Fork document variants from the VariantBar
5. Bundle export &rarr; whole family in one pass

### I want to reuse parts across projects
1. Draw a shape, select it with `M`
2. Promote to part (Library panel)
3. In any project: Library &rarr; select part &rarr; stamp mode
4. Save related parts as a Pack for sharing

## Sample Content

GlyphStudio ships with starter content to get you going:

| Starter | Type | What it provides |
|---------|------|-----------------|
| Loop Starter | Template | 32x32, 4 frames, animation-ready |
| Character Variant Starter | Template | 32x48, Warm/Cool palette variants |
| Palette Essentials | Pack | Warm and Cool palette shift variants |
| Shape Basics | Pack | Circle, diamond, and arrow stamps |

These appear in ProjectHome automatically on first launch.

## Key Shortcuts

| Action | Shortcut |
|--------|----------|
| Pencil | `B` |
| Eraser | `E` |
| Fill | `G` |
| Marquee select | `M` |
| Swap FG/BG | `X` |
| Undo | `Ctrl+Z` |
| Redo | `Ctrl+Shift+Z` |
| Duplicate frame | `Ctrl+D` |
| Play/Stop | `Space` |
| Step frames | `Left/Right` |
| Library search | `Ctrl+F` |

The shortcut strip at the bottom updates based on what you are doing.
