# Visual Critique Rubric

Every sprite revision gets scored against these 7 dimensions.
Score each 1–5. The sprite ships when every dimension scores 3+.

## 1. First-Glance Read (what does this look like in 1 second?)

- 5: Instantly reads as the intended subject
- 3: Reads correctly after a moment of focus
- 1: Reads as something unintended (blob, creature, furniture)

## 2. Silhouette

- 5: Silhouette alone identifies the character type
- 3: Silhouette is humanoid but could be anything
- 1: Silhouette is ambiguous or broken

## 3. Proportion

- 5: Head, torso, limbs feel balanced for the style
- 3: One area feels off but doesn't ruin the read
- 1: Head/body ratio or limb sizes feel wrong

## 4. Pose / Stance

- 5: Pose feels alive, weight is convincing
- 3: Pose is static but not broken
- 1: Pose is robotic, stiff, or anatomically impossible

## 5. Value Separation

- 5: Light direction is clear, forms read in grayscale
- 3: Some value contrast but flat in places
- 1: Flat, muddy, or values fight the forms

## 6. Material Cues

- 5: Metal reads as metal, skin as skin, cloth as cloth
- 3: Materials are hinted but not distinct
- 1: Everything looks the same material

## 7. Noise / Cluster Quality

- 5: Every pixel serves the form, no stray singles
- 3: Mostly clean, a few noisy areas
- 1: Scattered single pixels, broken shapes, visual static

## Critique Template

When critiquing a sprite, answer these in order:

1. **What does this read as?** (one sentence, first glance)
2. **What reads wrong first?** (the biggest problem)
3. **Top 3 changes** that would improve it most
4. **Silhouette test:** Does it work with all interior removed?
5. **Score:** Rate each of the 7 dimensions (1–5)
